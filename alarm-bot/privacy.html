Alarm Bot 개인정보 처리방침

최종 수정일: 2025년 8월 4일

1. 개인정보 수집 및 이용 목적

Alarm Bot은 다음과 같은 목적으로만 개인정보를 처리합니다.

  • 개인화 알람 메시지 생성: 사용자 맞춤형 AI 음성 메시지 제공
  • 날씨 정보 제공: 사용자 지역의 현재 날씨 정보
  • 일정 통합: 캘린더 일정을 포함한 알람 메시지

2. 수집하는 개인정보 항목

사용자 입력 정보 (로컬 저장)

  • 프로필 정보: 이름, 성별, 생년월일
  • 관심사: 뉴스 카테고리 선택
  • 알람 설정: 알람 시간, 제목, 반복 설정

일시적 처리 정보

  • 위치정보: AI 메시지 생성 시에만 사용 (즉시 삭제)
  • 캘린더 일정: AI 메시지 생성 시에만 사용 (즉시 삭제)

3. 개인정보 처리 방법

로컬 저장 원칙

  • 모든 개인정보는 사용자 기기에만 저장
  • 외부 서버에 개인정보 저장하지 않음
  • SQLite 데이터베이스로 암호화 보관

외부 전송 최소화

  • AI 메시지 생성 시에만 필요한 정보를 일시적으로 전송
  • 전송된 정보는 메시지 생성 후 즉시 삭제
  • 사용자 식별 정보는 전송하지 않음

4. 제3자 서비스 연동

Google Gemini AI

  • 목적: 개인화 알람 메시지 생성
  • 전송 정보: 이름, 성별, 날씨 정보, 일정 정보 (익명화)
  • 보관 기간: 즉시 삭제 (저장하지 않음)

기상청 API

  • 목적: 날씨 정보 조회
  • 전송 정보: 위치 정보 (시/군/구 단위)
  • 보관 기간: 전송하지 않음 (API 조회만)

SBS 뉴스 RSS

  • 목적: 뉴스 정보 조회
  • 전송 정보: 없음 (공개 RSS 조회)

5. 개인정보 보유 및 이용 기간

  • 앱 사용 중: 기기 내에서만 보관
  • 앱 삭제 시: 모든 데이터 자동 삭제
  • 외부 전송: 전송하지 않음 (AI 처리 시에만 일시적 사용)

6. 사용자 권리

개인정보 관리

  • 앱 내 설정에서 언제든지 수정/삭제 가능
  • 위치 접근, 캘린더 접근 권한 언제든지 철회 가능
  • 특정 기능(날씨, 뉴스, 캘린더) 비활성화 가능

7. 개인정보 보호 조치

  • 기술적 조치: SQLite 암호화, HTTPS 통신
  • 최소 수집: 필요한 정보만 로컬 저장
  • 즉시 삭제: 외부 처리 후 즉시 삭제
